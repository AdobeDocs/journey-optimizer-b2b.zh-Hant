# Adobe Journey Optimizer B2B Prime檔案

[!DNL Adobe Journey Optimizer B2B Prime]是AI優先的行銷自動化解決方案，可讓行銷人員和業務開發代表規劃方案、編排個人化個人歷程以及確認潛在客戶。 使用對話式AI來建立行銷活動、管理對象，並跨電子郵件和WhatsApp頻道傳遞內容。
