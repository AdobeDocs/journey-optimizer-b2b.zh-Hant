# 聊天介面

聊天面板已內嵌於整個[!DNL Adobe Journey Optimizer B2B Prime]。 在這裡，您可使用簡單語言與AI代理程式互動，以建立方案和歷程、建立和比較人員清單、調查銷售機會、設定評分等。

在左側導覽中選取&#x200B;_AI助理_&#x200B;圖示以開啟面板。 面板標題有四個控制項：

| 控制 | 說明 |
|---------|-------------|
| **新交談** | 開始全新的聊天（清除目前的執行緒）。 |
| **交談記錄** | 開啟過去的交談，以便重新開啟一個交談。 |
| **交換面板** | 將聊天面板移至工作區的另一側。 |
| **摺疊** | 隱藏面板，讓工作區有更多空間。 |

面板底部是訊息方塊，您可以：

* 新增訊息並按&#x200B;**Enter**&#x200B;以傳送（**Shift+Enter**&#x200B;插入新行）。
* 使用&#x200B;_附加_&#x200B;圖示附加檔案（支援的格式： `.txt`、`.md`、`.csv`、`.json`、`.xlsx`、`.docx`、`.pdf`）。 使用CSV和試算表上傳功能來開始匯入銷售機會。

## 詢問AI助理

有兩個同樣有效的方法可以完成工作 — 您永遠不需要使用斜線選單。

**自然語言** — 請以您對同事說的方式輸入您的要求：

* _「建立新試用註冊的歡迎歷程。」_
* _「為什麼john@acme.com沒有進入這個歷程？」_
* _「比較我的『Q3網路研討會出席者』與『企業帳戶』清單。」_

代理程式會將您的措辭與幕後適當的技能進行比對，並執行適當的工作流程。

**正斜線(/)功能表** — 輸入`/`以開啟助理可以執行的所有功能表。 當您想要探索功能或直接跳至已知的工作流程時，這會很有用。

## 斜線(/)功能表

在訊息方塊開頭或空格後使用`/`開啟功能表。 當您繼續輸入時，清單會依名稱、說明或ID進行篩選，例如`/journey`會縮小為歷程相關命令。

使用&#x200B;**/**&#x200B;來移動專案，**Enter**&#x200B;或&#x200B;**Tab**&#x200B;來選取，以及&#x200B;**Esc**&#x200B;關閉。 您也可以直接按一下專案。

功能表專案會分組到標籤區段中：

| 區域 | 包含 |
|---------|----------|
| **調查** | 診斷、唯讀查詢（例如潛在客戶調查、意圖查詢）。 |
| **驗證** | QA和稽核工作流程 |
| **組建** | 建立工作流程（計畫、歷程、清單、評分）。 |
| **匯入** | CSV銷售機會匯入。 |
| **其他** | 不屬於上述類別的專案。 |

功能表也會列出&#x200B;**聯結器** （例如，_瀏覽Marketo_&#x200B;會開啟選擇器對話方塊）和&#x200B;**導覽捷徑**，讓您跳至工作區畫面。

### 選取功能表專案

當您從功能表選取技能或代理程式時，會插入開始提示訊息方塊供您編輯 — 它&#x200B;**未**&#x200B;自動傳送。 例如，選取「_規劃行銷活動_」，在「_」中捨棄，並為「[」行銷活動名稱]規劃新的Marketo方案。 我將上傳簡報。&quot;_ 填入方括弧中的預留位置，然後按&#x200B;**Enter**&#x200B;進行傳送。

聯結器會開啟強制回應視窗，而非插入文字。 導覽捷徑會直接帶您前往工作區中的該畫面。

>[!NOTE]
>
>有些命令顯示為灰色，並標示為&#x200B;_即將推出_。 這些功能是由功能旗標所限定，而且尚未對您的帳戶啟用 — 選取其中一個則不會產生任何效用。 可用的集合取決於為您啟用的功能。

## 技能

技能是代理程式知道如何執行的封裝工作流程 — `/`功能表和自然語言請求背後的建置區塊。 每個技能都隨附逐步指示和一個工作所需的特定工具（例如「發佈歷程」、「比較兩個人清單」、「建立評分模型」）。

如需目前支援技能的完整清單，請參閱&#x200B;_[AI助理技能](./skills.md)_。

技能的重要須知事項：

* **技能為產品範圍。** 在[!DNL Journey Optimizer B2B Prime]中，您會看到各種產品專屬技能（歷程、人員清單、評分、頻道、傳送時間最佳化等）。 部分技能僅限[!DNL Marketo Engage]，部分技能同時適用於這兩種產品（銷售機會匯入、產品知識）。 您只會看到與您所在位置相關的技能。
* **您不需要記住技能名稱。** 說明您的目標，而代理程式會挑選相符的技能。 `/`功能表是通往相同工作流程的更快、可發現的捷徑。
* **部分技能僅供讀取；其他技能則會變更內容。** 調查和報告技能（例如，潛在客戶調查、意圖查詢、傳送時間報告）只會讀取資料。 建置和設定技能（例如歷程建立、評分）以建立或變更資料。

## 後續提示

助理回覆後，通常會根據您剛才的動作，顯示一列可點選的後續提示。例如，建立歷程後，可能會提供&#x200B;_「發佈此歷程」_&#x200B;或&#x200B;_「新增等待步驟」_。 按一下繼續，不輸入內容。 這些只是建議；您隨時都可以輸入自己的下一則訊息。

## 提示

* **使用識別碼來指定。** 在調查或編輯時，請包含電子郵件地址、人員名稱、歷程名稱或清單名稱，以便代理程式對正確的資產採取行動。
* **使用`/`來探索。** 如果您不確定助理可以做什麼，請開啟`/`功能表並略過類別。
* **編輯起始提示。** 選取技能可為您提供範本 — 在傳送前先取代`[bracketed]`預留位置。
* **請先上傳匯入。** 針對銷售機會匯入，請先附加CSV，然後說明您要對其執行的操作。
* **當您切換至不相關的工作時，請開始新的交談**，以便較早的內容不會影響新要求。
