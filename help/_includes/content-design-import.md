# 內容製作 — 匯入HTML

匯入的內容可以是：

* 包含內建樣式表的HTML檔案
* 包含HTML檔案、樣式表(.css)和影像的.zip檔案

  >[!NOTE]
  >
  >.zip 檔案結構沒有限制。 不過，參照必須是相對參照，而且符合.zip資料夾的樹狀結構。 影像會一律上傳至連線的Marketo Engage資產存放庫。 如果您想要在Experience Manager Assets中管理影像檔案，請在該應用程式介面中分別上傳這些檔案，並變更電子郵件設計空間中的影像連結。

若要匯入包含HTML內容的檔案(_T):_

1. 從&#x200B;_[!UICONTROL 設計您的範本]_&#x200B;首頁，選取&#x200B;**[!UICONTROL 匯入HTML]**&#x200B;選項。

1. 拖放包含 HTML 內容的 HTML 或 .zip 檔案，然後按一下「**[!UICONTROL 匯入]**」。
